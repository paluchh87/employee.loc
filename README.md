# employee.loc
Laravel 5.7
<br>jQuery/Ajax

<br><b>Задача:</b>
<br>Поработать с ajax и Laravel

<br><b>Установка:</b>
<br>composer install
<br>Подправить .env с доступом к БД
<br>php artisan key:generate
<br>php artisan migrate --seed
<br>Зарегистрировать своего пользователя на сайте
